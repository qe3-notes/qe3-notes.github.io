tag description
!DOCTYPE html Tells the browser we’re using HTML5
html lang="en" Root of the HTML document, the attribute tells the browser the document is in English
head Contains metadata about the webpage.
meta charset="UTF-8" Defines the character set as UTF-8, which supports almost all characters, including special symbols.
meta name="viewport" content="width=device-width, initial-scale=1.0" Ensures the page is responsive, meaning it adapts well to different screen sizes, like mobile devices.
title Defines the title that appears on the browser tab when you open a webpage.
body Contains all the visible content of the webpage.
h1 The most important and largest heading.
h6 The least important and smallest heading
p Paragraphs, blocks of text in a document,.
div A generic container element that doesn’t have any semantic meaning.
span A generic inline container.
em Used to emphasize text, which often appears italicized.
i Italicized text.
strong Used for important text, often displayed in bold.
b Bold text.
u Underlined text.
header Used to define the top section of a webpage or a section, typically containing a logo, navigation, or introductory content.
nav Represents a section of the page that links to other pages or sections, typically used for navigation bars.
main Represents the main content of the page. Each page should only have one!
article Used to represent a standalone bit of content, such as a blog post or news article.
section Represents a section of content within a page, which typically has a heading and groups related content.
footer Defines the footer for a section or an entire page. It typically contains information like copyright, links to policies, or contact information.
ul Unordered Lists. These create bulleted lists, where the order of the items doesn’t matter.
ol Ordered Lists. These create numbered lists, where the order of the items is important.
li Individual list item, used by both types of list.
a href="http://www.something.com" target="_blank" anchor tag linking to www.something.com that will open in a new tab
br A line break, used like hitting the enter keyt
